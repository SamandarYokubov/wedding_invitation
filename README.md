## Customization

All styling is in `styles.css`. You can easily customize:
- Colors (currently using gold/cream theme)
- Fonts
- Spacing and layout
- Background gradients

## Map Integration

The map uses Google Maps embed. For production use, you may want to:
1. Get a Google Maps API key
2. Replace `YOUR_API_KEY` in `script.js` with your actual API key
3. Use the embed API for better control and reliability
